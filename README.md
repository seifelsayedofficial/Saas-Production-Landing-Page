## ◼ Overview

FlowSync is a modern SaaS marketing landing page designed to:

• Communicate product value clearly  
• Showcase features and workflows  
• Present key metrics and testimonials  
• Display pricing tiers with interactive toggle  
• Provide a clean, conversion-focused structure

The project emphasizes performance, scalability, and visual hierarchy.

---

## ◼ Tech Stack

• HTML5  
• Tailwind CSS (CDN version)  
• Vanilla JavaScript  
• Alpine.js (lightweight state handling)  
• AOS (scroll animations)  
• Swiper.js (carousel system)  
• Lucide Icons

No heavy frameworks. No complex build setup required.

## ◼ Responsive Strategy

Mobile-first architecture using Tailwind breakpoints:

• sm → 640px+  
• md → 768px+  
• lg → 1024px+  
• xl → 1280px+  
• 2xl → 1536px+

Layout considerations:

• No horizontal overflow  
• Fluid grid transitions  
• Adaptive typography scale  
• Safe padding on small devices  
• Proper stacking behavior
